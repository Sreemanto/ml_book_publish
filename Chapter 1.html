Task 1 (1 mark)#

A file named Lincoln.txt is included in the folder containing this Notebook. Open the file and write some traditional Python code (one that uses nested for loops) to extract all the words from this file into a list named word_list. The list should only include the words from the file in lower case (punctuation, hyphens, new line characters etc should not be included). A portion of the output is shown below for your convenience:

Hint: You can use a very short snippet of code exploiting Python’s regular expression re module, which would make your task a whole lot easier.

import re
import pandas as pd

word_list = []

def lower(match_obj):
    '''
    This function is use to convert the uppercase character to lower case character
    '''
    char_elem = match_obj.group(0)
    if char_elem.isupper():
        return char_elem.lower()
    else:
        return char_elem.lower()
  



with open('./data/Lincoln.txt') as f:
    for line in f:
        line = line.replace("\n", "")
        
        #regex to change the worde to lower case
        new_line = re.sub(r"[a-zA-Z]",lower, line)
        
        #spliting the line in to words on the basis of " "
        words = new_line.split(" ")
        
        #looping through each word to remove the punctuation and copying them in a list
        for word in words:
            #regex to get rid of punctuations
            new_word = re.sub(r'[^a-zA-Z0-9\s]', '', word)
            word_list.append(new_word)

Task 2 (1 mark)#

Do the same you did in Task 1 above, this time around using a list comprehension instead.

word_list = []
with open('./data/Lincoln.txt') as f:
    for line in f:

        #regex to change the worde to lower case
        words = re.sub(r"[a-zA-Z]",lower, line.replace("\n", "")).split(" ")
        
        #list comprehnsion
        temp = [re.sub(r'[^a-zA-Z0-9\s]', '', word) for word in words]
        
        word_list.extend(temp)
